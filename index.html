Project Idea

Do-It-Yourselferator

Project Objective

The learning barrier for new people to crafts to know how they should start and proceed isn’t too high for the most part. However, when starting a new skill those first steps are daunting to anyone who wishes to begin the journey of learning in any form. This website will be able to address peoples curiosities towards crafting with ideas they may be interested in with instructions on how to make these items. The goal of the project is to build a site that can be the start of someones crafting journey. This will be done by generating randomly selected craft projects that can be done at home.

My Motivation

I have always enjoyed finding new do it yourself projects to do at home, and I see a lot of people searching for something that they can be proud of. The enjoyment of making anything yourself is fulfilling, you don’t just gain a new item but a new perspective on every object around you. You develop a new appreciation for the material items you poses. The entire process of crafting is special, you’re putting your passion into the creation of something new, and in my opinion that is the epitome of art and self-expression. If this project can assist others in finding their passions, it can be a part of making society a better place through self-development and creativity.

Tell Me More!

The website address will be “https://www.doityourselferator.com”, this will be the hub for do-it-yourself projects anyone can start regardless of skill level. I have chosen this name as I see the product provided to be a generator of inspiration for the users who visit. When you arrive on the home page you will be greeted by a minimalist designed theme that makes the user feel comfortable. Personally I believe the design choice is one of the bigger focuses for this project. Many websites these days have too much information and options for the user from the first page, it’s overwhelming and unfriendly for a new visitor to navigate and find what they want. By simplifying the options presented we can make the users feel like the website is a safe, inviting place to explore.

"You like style?" - Barry Bee

The style also supports the functionality of the website. As the reason for the site to exist is to inspire creativity through project ideas it needs to present the information in an easy to digest way. We can do that through making most of the visible space on the screen dedicated to presenting the projects and buttons to cycle through new ones until the user finds something that peaks their interest. The cycle function should be fluid and easy to use, we can use a well designed function built in that allows the website to pre-load a set number of projects as alternatives before the cycle option has been selected. This will allow the user to feel like the website is responding as quickly as possible to their requests. We don’t need to worry about load times for the main page when first opened due to all other features on the page being as simple as possible, and therefore less data required to load all assets. Functionality as a feature I believe has become a huge factor for any website competing in the last 5 years of IT from the users perspective.

But Wait... There's More!

We are getting more and more used to everything working “as it should” as soon as we start any piece of software, and this project will be very focused on user experience as something we give to every person who adventures into this website. One of the hurdles we will face within this project will be incorporating the projects themselves into the website display function. Though I posses numerous books of projects myself that are free to be used as options on this website, I worry that it won’t be enough looking into the future. If we are to build a piece of software that can scour the internet for ideas, convert them into a form that the website will present to the user I can see a level of longevity for new ideas to present to the user base. Long term I see the website being incredibly popular, current trends from Generation Z are leaning heavily into the idea of self expression and discovery. Though the website will be accessible to any age range, the audience I think it will connect to heavily will be the younger generations. I personally discovered my passion for these creative works through the internet as a young teen, and when your education about a lot of the world comes through the internet, we can teach people things they wouldn’t get to experience through normal streams of learning.

Tools!?!

From the tools and technologies perspective there is only a few major pieces of technology to develop for this project to excel. High level web design is possible with many tools, we will use Atom and Sublime for creation of the website to achieve the ambitions set. The programs to data grab and organise do-it-yourself projects will also be built using the last two suggested text editors. I see this to be a challenge but not something to be concerned about from the start, as this will take a considerable amount of time compared to the rest of the project to be developed and featured. Lastly we will use a host platform for the website with an excellent track record for up time, while also giving us the options to increase bandwidth with ease as the user base grows.



Le Skille, What the Hell is That!?!

An excellent level of skill in web design, eventually having a full time web developer would be optimal as they could continually work on making the project as advanced and up to date with the ever changing internet landscape. We also will need the skills in developing a web crawler and data conversion tool that would allow the website to keep adding new projects throughout operation. These positions we will need people who are skilled in HTML for the web design side of the project. As for building the pieces of software needed for obtaining and converting data, Python is an excellent language for building this software.

And what will the outcome be?

The outcome we are aiming to achieve with this project is a website that gives its users the knowledge and inspiration to build creative projects that are brought to them through the intuitive web design. By setting the goal of this project to lower and potentially remove the first hurdle into do-it-yourself projects I hope that it inspires a larger community of people who can embrace their inner artistic side and explore the human ability of creation. Broadly speaking, I want this project to be about enabling others to discover a new part of themselves, a part that makes us both proud of what we have, and can achieve. It’s lofty in its ambitiousness, but I think ideas that create real positive changes in our communities are always going to start in small obscure ways.
